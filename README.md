# user-prefs
Examples of "platform-wide" preferences for `harness+model` in agentic workflows.

## Claude AI 

- The ecological `claude.md` user-pref incorporates learnings from [Karpathy Style skills](https://github.com/forrestchang/andrej-karpathy-skills/blob/main/CLAUDE.md) for Claude models.
- The `memory-enablement-checklist.html` is a way to ensure "memory hygiene" across projects when dealing with parallel agentic workflows. Doing technology development with natural language based programming requires such abstractiosns for economic "memory management".
- The `trusted-hosts.md` is an allow-list for egress control, as agents can attempt to eagerly download data from URLs other than those in package managers, especially if they have been given wider and unchecked permissions for orchestrating agentic workflows. Try to mitigate cyber attacks via compromised supply chains, to the extent possible.
